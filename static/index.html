CSV email domain counter

This application goes through a list of emails and returns a sorted list of all the domains sorted by which one has the most emails related to it. The data input format is a CSV file that has an 'email' column

example CSV:
first_name, last_name, email, ip_address
Alice, Alison, alice@alisonblog.com, 192.168.1.1
Bob, Brown, bob@gmail.com, 192.168.1.1
